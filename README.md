# UniversityManagementSystem


## Introduction
This University Management System is a Java-based application designed to simplify the management of university-related processes. It utilizes AWT (Abstract Window Toolkit) and Swing for its graphical user interface. The system allows for adding and managing faculty and student information, updating personal and academic details, and viewing fees and academic results.

## Features
- **Faculty Management:** Add new faculty members, update their information, and manage their profiles.
- **Student Management:** Enroll students, update student information, and handle student academic records.
- **Fees Management:** View and manage fee structures for different programs and student categories.
- **Results Management:** Record and display academic results for students.

## Requirements
- Java Development Kit (JDK) - Version 8 or above
- Any Integrated Development Environment (IDE) that supports Java (e.g., IntelliJ IDEA, Eclipse)

## Installation and Setup

1. Clone the repository to your local machine:
2. Open the project in your preferred Java IDE.
3. Ensure that JDK is properly installed and configured on your IDE.
4. Build and run the application.
